어떤 플랫폼으로 구현할까요?

현장에서 스마트폰으로 사용하는 앱 — 실행 환경을 선택해주세요

A

React PWA (웹앱)

브라우저에서 실행, 홈 화면 추가 가능. 설치 불필요, 배포 간단. 오프라인 지원(로컬스토리지). 개발 속도 가장 빠름.

장점

  • 즉시 개발 시작 가능
  • 와이어프레임(HTML)과 동일 스택
  • PC/태블릿에서도 동작

단점

  • 카메라 접근 제한적(모바일 브라우저)
  • 네이티브 앱보다 UX 차이
B

React Native (네이티브 앱)

iOS/Android 네이티브 앱. 카메라·갤러리 완전 지원, 네이티브 UX. Expo로 빠른 개발 가능.

장점

  • 카메라/갤러리 완전 지원
  • 오프라인 완전 지원
  • 앱스토어 배포 가능

단점

  • 개발 환경 세팅 필요
  • 빌드/배포 절차 복잡
C

React 모바일웹 (반응형)

PWA 기능 없이 순수 반응형 웹. 현장에서 모바일 브라우저로 접속. 서버 없이 로컬 파일로도 실행 가능.

장점

  • 가장 단순한 구조
  • 별도 설치 없음
  • 빠른 프로토타이핑

단점

  • 오프라인 지원 없음
  • 데이터 영속성 주의
